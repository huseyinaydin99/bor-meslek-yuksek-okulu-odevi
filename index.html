Falan filan bir plan dı dı dı dı diyooo
Wireless şifre kırmada herkesin zorlandığını biliyorum. Wireless şifresini kırmanın zor olduğunu sanıyorsunuz ama aslında çok basit yöntemi var. Buyrun : WEP şifreleme tekniğini bu defa Ubuntu işletim sistemi ile hackleyelim. Öncelikle bilgisayarınıza Wubi programını indirip kurun. Kurulum tamamlandıktan sonra bilgisayarınızı yeniden başlatın ve açılışta size soracağı seçeneklerden Ubuntu’yu seçin… Uygulamalar -> Uçbirim -> sudo apt-get install aircrack-ng yazın ve kurun..
